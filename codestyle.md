# 代码规范

基于 Google 代码风格指南

## Python 代码规范
- 使用 4 个空格缩进
- 行长度不超过 80 个字符
- 使用小写字母和下划线命名变量和函数
- 类名使用驼峰命名法
- 导入语句分组，标准库在前，第三方库在后

## JavaScript 代码规范
- 使用 2 个空格缩进
- 使用单引号
- 语句末尾使用分号
- 使用 const 和 let，避免使用 var
- 函数和变量名使用驼峰命名法
