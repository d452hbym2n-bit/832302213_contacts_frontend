# 通讯录管理系统 - 前端

## 项目描述
基于HTML、CSS和JavaScript开发的通讯录管理系统前端界面，实现前后端分离架构。

## 功能特性
- 添加联系人
- 编辑联系人信息  
- 删除联系人
- 联系人列表展示

## 技术栈
- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API

## 运行方式
1. 启动后端服务（端口5000）
2. 在frontend目录下运行：`python -m http.server 8000`
3. 访问 http://localhost:8000
